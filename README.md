
# Installation Instructions

## Manual Installation

1. Navigate to the plugin folder location on your computer.
2. Copy the entire folder containing the manifest.json and index.html files.
3. Locate your Photoshop Plug-ins directory. This is typically found at:
   - Windows: C:\Users\[Username]\AppData\Roaming\Adobe\UXP\Plugins\External\(create new folder and paste here)
4. Restart Adobe Photoshop.
5. Ensure Developer Mode is enabled:
   - Go to Edit > Preferences > Plugins.
   - Check the box for "Enable Developer Mode".
6. Open the plugin via the menu: Plugins > Kitsui Hyphenation > Kitsui Hyphenation.

## Usage

1. Type a word into the input field at the top of the panel.
2. The hyphenated result will be displayed instantly in the box below.
3. Syllable breaks are indicated by red dots for clear visibility.
